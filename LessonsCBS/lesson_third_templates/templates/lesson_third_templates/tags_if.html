Операторы ==, !=, <, >, >=, <= ,in

==

{% if x == "x value" %} {{x}} {% endif %}

!=

{% if x != "y value" %} x != "y value" {% endif %}

<

{% if value < 15 %} if value < 15 ->{{value}} {% endif %}

>

{% if value > 5 %} if value > 5 ->{{value}} {% endif %}

>= , <=

{% if value <= 10 %} if value <= 10 ->{{value}} {% endif %}

ELIF!

{% if var1 %} {{ var1 }} {% elif var2 %} {{ var2 }} {% elif var3 %} {{ var3 }} {% else %} EMPTY! {% endif %}

Equvivalent firstof!

{% firstof var1 var2 var3 "EMPTY!" %}

IN!

{% if "name" in obj %} {{obj.name}} {% endif %}
{% if "bc" in "abcdef" %} Это предложение выводится, поскольку "bc" является подстрокой "abcdef" {% endif %}
{% if "bc" not in "adas" %} Это предложение выводится, поскольку "bc" не является подстрокой "adas" {% endif %}
{% if "hello" in greetings %} Если greetings представляет собой список или множество, одним из элементов которого является "hello", это предложение будет выведено. {% endif %}
Операторы сравнения нельзя записывать цепочкой, как в Python или в математической записи. Например, вместо записи: if a > b > c
{% if a > b > c %} (WRONG) {% endif %}
следует писать: if a > b and b > c
{% if a > b and b > c %} 100 > 10 > 1 {% endif %}
